# Detection of Small Particles in Ocean Using SAR Images

## Project Overview

This project detects small particles in ocean environments using Synthetic Aperture Radar (SAR) images and the YOLOv8 object detection model.

## Technologies Used

- Python
- YOLOv8
- OpenCV
- Google Colab
- Roboflow

## Dataset

SAR image dataset annotated using Roboflow.

## Model Performance

- Precision: 93.9%
- Recall: 87.3%
- F1 Score: 90.5%
- mAP@50: 93.9%

## Workflow

Dataset Collection
→ Preprocessing
→ Annotation
→ YOLOv8 Training
→ Validation
→ Particle Detection
→ Result Visualization

## Future Scope

- Real-time ocean monitoring
- Marine pollution detection
- Oil spill analysis
